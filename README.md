# 🚀 차트만 보고 롱·숏 치는 - 코인선물 적성검사

단타 적성검사(day-trader-test)와 같은 구조의 바이럴 모의투자 게임. 코인명을 숨긴
**실제 바이낸스 5분봉** 차트로 20시간(240봉) 동안 롱·숏 선물 매매를 하고 적성 등급을
받는다. 최대 10배 레버리지 — 고배율 선물의 위험(강제청산)을 체감시키는 것이 기획 의도.

## 실행

```powershell
python -m http.server 8020 --directory public
```

브라우저에서 `http://localhost:8020`을 엽니다.

## 데이터 갱신

```powershell
node scripts/fetch-klines.mjs
```

바이낸스 현물 API에서 코인별 "역사적 명장면" 10일치(2880봉) 5분봉을 수집해
`public/data/{i}.json`으로 저장합니다 (총 34개 차트). 가격은 청크 시작가=100으로
지수화해 코인 정체를 숨기고, manifest.json에는 등급 힌트(메이저/알트/밈)만 노출됩니다.

## 게임 규칙 / 구조

- 가상자금 1,000만원 · 사전 20시간 공개 → 본게임 240봉
- 격리마진 · 전액 증거금 진입 · 레버리지 1/2/5/10/20/50배 (50배 청산 거리 약 -1.5%)
- 수수료 0.05% (진입/종료 각각, 명목가 기준), 유지증거금률 0.5%
- 청산가 = 진입가 × (1 ∓ 1/lev) / (1 ∓ MMR), 봉의 고가/저가로 청산 판정
- 청산 = 전액 소실 = 즉시 게임 오버, 등급 💀
- 등급은 "현물 존버 대비 알파" 기준 (SSS~F), 결과에 "10배 롱 존버였다면" 교육용 비교 포함
- 재생 중 주문은 다음 봉 시가 체결, 일시정지 중엔 현재 봉 종가 즉시 체결 (룩어헤드 방지)
- 친구 도전장(`?g=차트.시작&r=상대자산`), 결과 카드 저장/복사, 로컬 기록 대시보드

## 미반영 (결과 화면에 면책 고지)

- 펀딩비, 슬리피지, 호가 스프레드, 청산 수수료/보험기금 차감
